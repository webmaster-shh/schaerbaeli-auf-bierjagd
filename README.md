# Schärbäli auf Bierjagd

**Schärbäli auf Bierjagd** ist ein kleines humorvolles Browser-Game im Vereinskontext der **Gugge Schärbähufä Hergiswil**.
Das Spiel ist als Spassprojekt für den Verein und das nähere Umfeld entstanden und nicht primär für eine breite Öffentlichkeit gedacht.


## Über das Spiel

In **Schärbäli auf Bierjagd** muss der Spieler mit dem Schärbäli verschiedenen Hindernissen ausweichen, Bier sammeln und Wasser vermeiden.
Ziel ist es, möglichst viele Punkte zu erreichen und sich einen Platz im Leaderboard zu sichern.


## Spielziel

- Hindernissen ausweichen
- Bier einsammeln
- Wasser vermeiden


## Steuerung

Das Spiel kann direkt im Browser gespielt werden.

**Steuerung:**
- **Handy:** Tippen
- **Desktop:** Klicken oder Leertaste


## Spielregeln

- **+3 Punkte** pro Bier
- **-1 Punkt** pro Wasser


## Hindernisse und Objekte

### Hindernisse
- Verkehrsschilder
- Werbepfösten

### Sammelobjekte
- Bier

### Negative Objekte
- Wasser


## Features

Aktuell enthält das Spiel unter anderem folgende Funktionen:

- Spielbar auf **Handy und Desktop**
- Browser-basiertes Gameplay
- Punktesystem
- Highscore-Speicherung
- Top-10-Leaderboard
- Benutzername-System
- Performance-Warnung bei schwächerer Darstellung
- Offline-/Fehleransicht bei fehlender Datenbankverbindung
- Laufende Schwierigkeitsanpassung während des Spiels

## Highscore

Das Spiel enthält ein Highscore-System mit Top-10-Leaderboard.
Spieler können mit einem Benutzernamen spielen. Bestehende Namen können weiterverwendet werden, wobei der gespeicherte Highscore nur überschrieben wird, wenn ein besseres Resultat erzielt wird.
